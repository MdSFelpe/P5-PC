Nossa intenção com o projeto:

Criar um fórum personalizado, com identidade visual e estilo de arte baseado em um jogo eletrônico. O principal intuito do fórum é integrar usuários em temas cotidianos, hobbies, tendências e dentre outros tópicos. isso acontece através de abertura e interação em threads para os usuários. Nosso projeto tem o foco em 4 páginas principais: Página de login, para o acesso à página principal do fórum; caso seja o primeiro contato do usuário, temos uma página de cadastro para o preenchimento dos dados mais importantes. A principal página de acesso do usuário é a página do fórum, na qual toda a interação será feita, hospedando os diversoso temas e conversas do fórum. Por fim, a página da tabela dinâmica (banco de dados), onde constará o nome dos usuários, os ids e datas de entrada.

Construção do projeto html:

  • Criação da página de login do fórum
  • Criação da página de cadastro do fórum
  • Criação da página principal, utilizada para responder outros usuarios e adicionar thread de assuntos
  • Criação de página para os desenvolvedores com tabela de quantidade de usuários cadastrados

Referências do projeto:

  • Página de design e Modelo
  • Site de ensino Css
  • Video de montagem de projetos html e CSS
  • Video de aprendizagem Javascript
